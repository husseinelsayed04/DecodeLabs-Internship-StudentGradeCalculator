# 🎓 Student Grade Calculator (Java Project)

## 📌 Project Description
This is a simple Java console-based application that calculates a student's total marks, average percentage, grade, and pass/fail status based on multiple subjects.

The project is designed using Object-Oriented Programming (OOP) principles.

---

## 🚀 Features
- Input student name
- Enter marks for multiple subjects
- Calculate total marks
- Calculate average percentage
- Assign grades based on percentage
- Determine PASS / FAIL status
- Input validation (marks between 0–100)

---

## 🧠 Concepts Used
- Object-Oriented Programming (OOP)
- Classes and Objects
- Arrays
- Loops
- Conditional Statements (if-else)
- Methods
- User Input (Scanner)
